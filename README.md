# Homebrew Microwave

Homebrew tap for the [Microwave CLI](https://github.com/microwave-sh/microwave-cli).

## Install

```sh
brew tap microwave-sh/microwave
brew install microwave
```

Or in a single command:

```sh
brew install microwave-sh/microwave/microwave
```

## Updates

The formula is updated automatically when new releases are published to [microwave-cli](https://github.com/microwave-sh/microwave-cli). No manual steps are required to receive new versions — run `brew upgrade microwave` to update to the latest release.

## Source

See [microwave-sh/microwave-cli](https://github.com/microwave-sh/microwave-cli) for source code, issue tracking, and release notes.
